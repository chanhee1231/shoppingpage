# Java-Project-ShoppingMall  
📂 **옷 쇼핑몰 사이트**

## 📌 프로젝트 소개  
Java와oracle을 활용한 간단한 **옷 쇼핑몰 웹사이트**를 구상하고 제작했습니다.  




---



## 🕰️ 개발 기간  
**2025.06.06 ~ 2025.06.09**

---

## ⚙️ 개발 환경  
- Java 21  
- JDK 21.0.7
- IDE
- Oracle

---

## 🔑 주요 기능

### 👤 로그인  
- 입력한 아이디를 기준으로 **관리자**와 **일반 사용자**를 구분합니다.

### 📝 회원가입  
- 이름, 아이디, 비밀번호를 입력하여 회원 등록이 가능합니다.

## ▶ 로그인 페이지
  
#### 1. 로그인 
- 아이디, 비밀번호를 입력하면 데이터베이스안에 데이터와 비교해서 로그인하는 기능

#### 2. 회원가입 
- 아이디, 비밀번호, 회원이름를 입력하여 회원가입하는 기능

#### 3. 프로그램 종료 
- 프로그램 종료기능

## ▶ 사용자 페이지
<img width="500" alt="image" src="https://github.com/user-attachments/assets/333f1133-6351-45b7-9ed0-923dcf85bd07" />

#### 1. 상품 목록 출력
- 상품 리스트를 보여주는 기능입니다. 
  
#### 2. 장바구니 확인
- 상품선택으로 장바구니에 등록한 정보를 리스트로 출력하는 기능
  
#### 3. 장바구니 담기
- 상품 리스트를 보여주고 장바구니에 추가하는 기능

#### 4. 장바구니 삭제
- 선택한 장바구니 정보를 리스트로 출력하고 원하는 상품을 골라서 삭제하는 기능

#### 5. 장바구니 초기화
- 해당 사용자의 장바구니를 초기화하는 기능

#### 6. 쿠폰 목록 확인
- 상품에 적용할 쿠폰의 목록을 확인하는 기능

#### 7. 쿠폰 목록 확인
- 장바구니에서 결제할 항목을 선택하고, 쿠폰 사용 여부를 묻고 쿠폰 적용하여 결제가 되는 기능


## ▶ 관리자 페이지
